# Commit and Push

> Purpose: Determine if work was done and push changes to remote
> Scope: Git staging, committing, and pushing

## Determine if work was done

After completing all assigned steps, check:
- Did you respond to PR comments?
- Did you route any issues (intake)?
- Did you implement any TODOs?
- Did you make any commits?

If no work was done (no comments to respond to, no issues to route, no TODOs implemented), skip ahead to the summary step.

## Push your work

If work was done:
1. Ensure all changes are committed with clear, concise commit messages
2. Push your branch:
   ```bash
   git push origin "$AGENT_BRANCH"
   ```
